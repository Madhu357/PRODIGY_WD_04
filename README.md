# PRODIGY_WD_04

## Weather App

This project is developed as part of my Web Development Internship at Prodigy InfoTech.

## Project Description

A weather application that fetches real-time weather data from an API based on the user's searched location. It displays important weather details like temperature, weather conditions, humidity, and wind speed.

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API

## Features

- Search weather by city name
- Displays current temperature
- Shows weather condition
- Displays humidity and wind speed
- Responsive user interface
